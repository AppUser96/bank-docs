# Настройка аккаунта Nova Bank

Перед началом работы в приложении Nova Bank необходимо заполнить данные вашего профиля. Внимательно следуйте инструкциям на экране.

> [!IMPORTANT]
> Все вводимые данные должны точно совпадать с информацией в ваших официальных документах (например, паспорте). Это необходимо для безопасного проведения операций.

1.  **Введите вашу почту**
    *   В поле ввода укажите адрес электронной почты, который вы использовали при регистрации (например, `www@example.com`).
    *   Проверьте правильность введённого адреса и нажмите **Далее**.

2.  **Выберите тип учётной записи**
    *   Выберите подходящий тип учётной записи из выпадающего списка.
    *   Нажмите **Далее**.  

    <img src="../img/Account Set Up Email 2.png" alt="Экран выбора типа учётной записи" width="300">

3.  **Добавьте адрес регистрации**
    *   **Адрес**: Введите ваш полный адрес регистрации.
    *   **Город или регион**: Укажите город или регион.
    *   **Почтовый индекс**: Введите почтовый индекс.
    Нажмите **Далее**.

    <img src="../img/Account Set Up Address.png" alt="Экран ввода адреса регистрации" width="300">

4.  **Введите личные данные**
    *   **Полное имя**: Введите ваши фамилию, имя и отчество.
    *   **Дата рождения**: Укажите дату в формате `ДД.ММ.ГГГГ` или выберите её из календаря.
    *   Нажмите **Далее**.

    <img src="../img/Account Set Up Personal Info 1.png" alt="Экран ввода личных данных с календарём" width="300">

Настройка аккаунта завершена. Теперь вы можете пользоваться всеми функциями приложения Nova Bank.
